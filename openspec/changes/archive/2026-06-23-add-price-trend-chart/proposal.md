# 新增首發價格趨勢折線圖與價格紀錄編輯

## Why

目前系統可以新增、查詢與顯示遊戲首發價格資料，但使用者主要是透過表格查看資料，不容易直接觀察近 20 年第一方大作首發價格的變化趨勢。

為了讓本專題更符合「價格變化觀察」的主題，需要新增折線圖功能，將每筆資料的發售日作為 X 軸，首發價格作為 Y 軸，讓使用者可以直觀看出價格隨時間的變化。

此外，使用者手動輸入資料後，如果發現日期、價格或文字欄位有誤，目前只能刪除後重新新增。為了讓資料維護更方便，需要提供既有價格紀錄的編輯功能，讓使用者可以直接修正資料並同步更新表格與圖表。

## What Changes

- 新增首發價格趨勢折線圖
- 使用 `release_date` 作為 X 軸
- 使用 `price` 作為 Y 軸
- 圖表資料來源為 SQLite 中的價格紀錄
- 圖表點位顯示遊戲名稱、平台、發售日與價格
- 沒有發售日或價格的資料不顯示在圖表中
- 新增 `PUT /api/prices/:id`，可修改既有價格紀錄
- 編輯資料時沿用新增資料的日期與價格驗證規則
- 前端表格新增編輯操作，使用者可修改既有資料
- 修改完成後重新載入表格與折線圖

## Impact

- 影響前端頁面顯示
- 會使用現有 `GET /api/prices` API 取得資料
- 會新增 `PUT /api/prices/:id` API
- 不需要修改資料庫結構
- 不影響原本新增、查詢、刪除與 optional crawler 功能
